About project:

On this page you can find various information regarding the Ancient Minecraft Launcher project.

There will be several Blocks here: Technical information, Roadmap, Car information and some thoughts of the author

What kind of Project is this? What is it made of and why is it needed?

 

The old original Minecraft Launcher has long changed graphically, functionally, and has long been redesigned and has become completely autonomous (Previously, the Website and Launcher were used).

 

And since everything has been updated and redesigned for a long time, the old versions of the Launcher are no longer supported.

Install them and you won't be able to play them.

 

This project was created in order to recreate the "Same" Launcher, add support to it, the ability to install new versions, immerse the user in "That time", as well as simply provide the opportunity to quickly launch the game, and "Give an alternative".

 

If someone suddenly wanted to remember "That Launcher", Or someone has a weak laptop/computer that loads Windows as much as possible, or new players would like to touch the "Story" at least that way, or you would like to play Minecraft on the old Launcher, but Mojang and Microsoft no longer support it. older versions of the launcher, the Ancient Launcher solves this problem.

 

The project was written in Python, the code was written as simplistically as possible, so as not to overload the system even on weak computers or Laptops.

 

The project will recreate the Old launcher and be maintained, as well as updated and improved in the future.

 

What plans for the future?

 

Here is a small express plan for future projects.:

 

Implement a modified file system and make it friends with the real one to support Modding.

 

Create a supply of modified versions for older versions of the game (For example Forge 1.5.2) and add a simplified one

 

Add all modern Launcher functionality to Ancient Launcher

(Changing Skins, changing Cloaks, creating Builds, and more). And also integrate it without violating the integrity of the 2013-2014 Launcher style.

 

Add support for Themes and Forms. For example, in the settings, you can set the theme of the launcher 2010, and when you launch the same launcher, its appearance will match the Launcher Minecraft 2010.

Or you can change the Shape of the window in the settings, for example, being on Windows 10, the shape of the launcher window will

have a Windows Aero Theme (Windows 7), or a Windows XP theme.

 

Also make 2 Launcher branches (Both will be supported independently of each other)

Where 1 Launcher will be the Classic launcher with the appropriate style.

and the second one is "From a parallel universe", that is, if Mojang or Microsoft hadn't radically changed the look and style of the Launcher so much, and would have continued to update it in the classic style.

 

There are many more plans. If the community supports the project and this Launcher, then everything will develop.

The launcher was created because while everyone is discussing whether the new versions are bad and which old ones were good against the background of new updates, the author himself preferred to continue playing on the old versions. And at some point, when it was thrown out of the official launcher again, I wanted to play on the Old version of Minecraft launcher. I couldn't download it, it doesn't have support for newer versions, and there are a lot of problems with this....

 

It was easier to create a new launcher with the look of the old one. That's exactly what was done, and this project/launcher is the result of such an "action". Everything was done from memory, as well as with an emphasis on the found screenshot of the version that the author remembered. This screenshot served as a reference during the development process (Memory is good, but it fails)
