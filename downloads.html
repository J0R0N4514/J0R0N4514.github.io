Launcher Features

 

+ Authorization of the player by Nickname (Choosing a nickname to launch the game)

 

+ Support for community servers (Unofficial servers)

 

+ Manual input of the game version (For advanced users)

 

+ Settings window

 

+ Display the latest project news with this Launcher

 

+ The ability to configure whether the Launcher will close after the end of the game

(According to the author's recollections, the launcher window closed as soon as the main game was closed)

 

+ Regulation of the possibility of automatic storage for the next login (Whether to fill in the Nickname and Version fields when launching the launcher)

 

+ Support for installing Old and New versions of the game simultaneously from the Mojang database.

 

+ Official and valid links to Minecraft sites are available (Minecraft.net , Bug Tracker and others)

 

+ Old Mojang games "COBALT" and "SCROLLS" have been found and added to the launcher menu

 

+ Running the game Offline (If you installed the Game, but then lost your internet connection)

Download Archived versions

 looking for a specific version but it's not on the list?

Perhaps these versions were not created by an official author, which is why they do not exist in this list.

Or it is possible that these versions were distributed in closed testing, and were leaked. Such versions are not included in this list because they were not going to be distributed to a wide audience.

 

 

Have found the desired version that can no longer be downloaded?

Apparently, there were problems with these versions of the launcher, because of which their distribution was stopped, or they contained critical vulnerabilities or serious problems. They are still listed as officially existing, but they can no longer be downloaded.

(only if you roll back the time)

 

The use of such versions is either an exclusive opportunity for specific people who still have them, or it is strongly discouraged. Depending on what is indicated for a particular version as the reason for the inability to download this version officially.
