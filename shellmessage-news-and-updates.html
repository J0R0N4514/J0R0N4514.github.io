22.07.2025 - Обновление

Новое обновление 1.7!

Вот список изменений:

 

+ Переработан интерфейс

+ Переработана структура кода

+ Добавлено больше пунктов в меню

* такие пункты меню как "настройки" и "Сервер" временно отключены. Будут добавлены темы

+ Добавлено Шифрование

+ Теперь доступно для скачивания

+ Обновлен Utility Server. Теперь он имеет собственный интерфейс, и не отображается в консоли

+ Utility Server теперь полноценное .exe приложение, а не запускаемый скрипт

+ Добавлена возможность отправки сообщения с помощью Enter

+ На сайте добавлена возможность скачивания Клиента и Сервера

+ Окна Личной и Основной информации теперь отображаются корректно всегда

+ Добавлена возможность вставки Эмодзи

+ При получении и отправке сообщения теперь присутствует звуковое сопровождение (Только на Windows, будут обновления)

+ Добавлена левая панель с информацией о текущем сервере.

+ Добавлена возможность переключения/смены сервера не выходя из программы

+ Добавлена возможность смены ключа шифрования

+ Переработано окно подключения к серверу

+ Теперь сообщения Отправителя не Дублируются

+ Сервер теперь может сохранять Лог-файлы, и имеет кнопку отключения и выхода

+ Заложены основы под будущий функционал

+ Исправлен баг со значками окон
