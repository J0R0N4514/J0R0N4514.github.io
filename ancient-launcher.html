About project...

Ancient-Launcher - Is An Unofficial Minecraft Launcher.

The prototype of this launcher is the Official Minecraft Launcher sample from about 2012-2014.

 

The official Minecraft launcher has changed a long time ago. But the old versions, with all the desire, can no longer be

installed and used, or used with restrictions, for example, the old launchers do not support

newer modern versions, and no one will ever provide support for the old launchers.

 

 

The task of Ancient Launcher is to recreate the good old launcher from 2013-2014, and provide full support and functionality even with new versions of the game and immerse the user in a time when

the official minecraft launcher was simpler, more minimalistic, and so on...

 

Perhaps you're reading this version, or maybe you've caught it? Then maybe you should download it.

This software is distributed free of charge and on an "As is" basis. Also, we recommend downloading the Launcher only from this, the official website of the project.

This application was created based mostly on Memory, more precisely on how the Author of the Launcher remembers it. And also based on the few screenshots found on the Internet, what this Launcher looked like at that time.

 

 

Unfortunately, since the Author has been using this launcher for a very long time, it was not possible to remember and find a video where the settings window would appear, or an overview of this version of the launcher. So treat this work not as a Reworking and inclusion of support in the launcher itself, but as a project aimed at Recreating that version of the Launcher.

What plans for the future?

 

This launcher will be updated further over time.

 

In the future (Perhaps, if possible), there will be support for official MOJANG and MICROSOFT Accounts, greater consistency with the original, redesign of the Launcher's interfaces and functionality, including its capabilities and support for versions on macOS and Linux, and performance testing on Older versions of Windows (XP / 7 / 8.1), Implementation of proposed ideas from the Community. As well as support for Launcher Styles.

 

For example, this is the 2014 style, and in the settings you can switch the style to the 2011-2010 image.

 

It is also possible, if possible,

that we will use the Windows Aero theme and window shape from Windows 7, which will also be available in the settings.

 

Support for Mods on older versions - Now, it will no longer be possible to install mods for Minecraft, for example on 1.5.2

, since the file system of the Mojang Launcher has changed, and mods for older versions are no longer supported.

We will try to bring back support and the ability to install mods on older versions of the game.
